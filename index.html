Media House

This is a media streaming app that I created using Svelte, Go, and mySQL. This was a personal project that allowed me to set up my own home media server. I used Svelte for the frontend, Go for the backend, and mySQL for the database. The app allows you to upload media files and stream them from anywhere on your home network. The app is still a work in progress, but I wanted to set up a media server for myself and this was a fun way to do it while learning new technologies. A screenshot will be coming soon for this, but the interface is similar to the Invoicer app. Take a look here

Invoicer

This is an invoicing app that I created using React.js, C#, and mySQL. This was a project I ended up working on for G-Full to manage invoices easily in one place.
